# Toll Consulting Plugin

Toll industry expertise packaged for Claude Code — operations consulting, procurement advisory, and structured document review.

## What's Included

### Agents

| Agent | Domain |
|-------|--------|
| Toll Consultant | Tolling operations, CSC systems, interoperability, enforcement, regulatory frameworks |
| Toll Procurement Consultant | RFP development, vendor evaluation, proposal analysis, contract management |

Both agents run on Sonnet and are invoked proactively when Claude detects toll-related work.

### Skills

**Document Review** — Guides structured review of toll project documents from an owner's representative perspective. Covers submittals, test plans, compliance artifacts, ICDs, and more. Classifies documents first, then applies category-specific review frameworks with severity-rated comments and approval recommendations.

## Installation

```
/plugin install bloknayrb/claudestuff toll-consulting
```
